# Projeto Integrado 
Tendo em vista a necessidade da empresa entendemos que uma das possibilidades de mitigar as falhas mencionadas sem impactar/elevar tanto os custos é usando o Load Balancer (Balanceador de Carga) onde o sistema redireciona automaticamente o fluxo de solicitações, dividindo as entre as máquinas disponíveis e se usássemos o Least Connections esse redirecionamento seria priorizado para os servidores com menos uso/demanda. Como complemento podemos ativar o recurso de Auto Scaling para que no caso do uso chegar a ficar muito alto será aberta uma nova instância/servidor para que possa suprir essa demanda de maneira rápida evitando que um servidor caia por alta demanda e também o recurso seja liberado numa necessidade, já que caso a demanda caia o próprio sistema irá diminuir a quantidade de servidores ativos, automaticamente diminuindo o valor a ser pago
